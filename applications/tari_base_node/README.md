# Tari base node

## Installation

### Prerequisites

#### Linux
```
sudo apt-get install git curl build-essential cmake clang pkg-config libssl-dev libsqlite3-dev sqlite3
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

```
### From source

```
cargo install tari_base_node
```
